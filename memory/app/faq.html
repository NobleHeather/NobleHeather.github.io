Foire aux questions

Je cherche une carte précise sur la page d'accueil

En faisant control + F, on ouvre la barre de recherche, ensuite il n'y a plus qu'à taper le nom ou une partie de l'indice !

Je voulais modifier/effacer une carte mais je n'arrive pas à la trouver

Il faut la rechercher avec le nom précis, sans faute d'orthographe, avec le même nombre d'apostrophe, de majuscule, d'espace... Dans le doute, chercher le nom sur la page d'accueil et faire un copier-coller

J'ai bien enregistrée ma carte mais l'image n'apparaît pas

Est-ce que l'image est bien dans le dossier IMAGES ? Est-ce qu'elle a bien été enregistrée avec le même nom dans le dossier et sur la page pour ajouter des cartes ? Est-ce que l'extension est bien écrite en toutes lettres aux deux endroits : oiseau.jpg ou oiseau.png ? Est-ce que c'est une extension inhabituelle comme .bmp ou .webp ? Dans ce cas peut-être qu'il faudrait essayer avec une autre image
